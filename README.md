# 🏀 Player Statistics Tracker

A simple desktop application built with Python and Tkinter to record and display player statistics in real time.

## 📸 Preview

> A clean GUI window where you can input player data and instantly see the results listed below.

## 🚀 Features

- Add player statistics manually through a graphical interface
- Tracks: **Points**, **Passes**, **Errors**, and **Blocks**
- Displays all entries in a scrollable text area
- Clears input fields automatically after each entry

## 🛠️ Built With

- **Python 3**
- **Tkinter** – standard Python GUI library (no extra installs needed)

## ▶️ How to Run

1. Make sure you have **Python 3** installed on your machine.
2. Clone this repository:
   ```bash
   git clone https://github.com/your-username/player-statistics.git
   ```
3. Navigate to the project folder:
   ```bash
   cd player-statistics
   ```
4. Run the application:
   ```bash
   python players_statistics.py
   ```

## 📋 How to Use

1. Enter the player's name in the **Player Name** field.
2. Fill in the stats: Points, Passes, Errors, and Blocks.
3. Click **"Add Statistics"**.
4. The player's stats will appear in the text area below.
